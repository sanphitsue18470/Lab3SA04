============== สรุปสิ่งที่ทำ ==============
เป็น mobile application ที่รวบรวมจังหวัดต่างๆสามารถดูสภาพอากาศได้เช่น อุณหภูมิ และ ความชื่น  เป็นต้น

commit 1 : เพื่มรูปภาพแต่ละจังหวัด
commit 2 : ตั้งค่าปุ่มกดชื่อจังหวัดให้อยู่ตรงกลาง
commit 3 : เปลี่ยนสีหัว Home กับ ชื่อจังหวัด
commit 4 : เปลี่ยนสี zipcode 
commit 5 : เพิ่มความดันอากาศจาก API JSON มาใส่ในแต่ละจังหวัด
